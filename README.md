## Ⅱ. 프로젝트 개요
- **프로젝트명**:  
  **MyPetTrip — AI 챗봇 기반 반려동물 장소 추천 서비스**

- **프로젝트 기간**:  
  2025년 9월 ~ 2026년 6월  

- **기획 목적**:  
  반려동물 동반 여행 시 발생하는 정보 분산, 조건 불확실성, 실제 이용 가능 여부 확인의 어려움을 해결하기 위해,  
  AI 기반 여행 상담(Chatbot)과*위치 기반 서비스(LBS)를 결합하여  
  반려동물과 보호자 모두에게 적합한 여행 정보를 제공하는 것을 목표로 한다.

---

## Ⅲ. 주요 기능

### 1. AI 기반 여행 상담 챗봇
- OpenAI GPT API 활용  
- 자연어 기반 질의 응답 지원  
- 예시:
  - “대형견 가능한 카페 추천”
  - “조용한 산책 코스 알려줘”

### 2. 조건 기반 필터링
- 대형견 가능 여부  
- 실내 / 실외 동반 가능  
- 소음 환경  
- 계단 및 접근성  

### 3. 반려동물 정책 데이터 구조화
- 공공데이터 및 크롤링 데이터 활용  
- 세부 정책 태그화:
  - 체중 제한  
  - 견종 제한  
  - 동반 가능 구역  

### 4. 커뮤니티 및 후기
- 사용자 리뷰 및 사진 제공  
- 실제 이용 경험 기반 정보 보완

## Ⅳ. 시스템 구조

### 1. Frontend (Flutter)
- iOS 플랫폼 개발  
- 챗봇 UI 및 지도 기반 탐색 UI 제공  
- Google Map API 활용  

### 2. Backend (Firebase)
- Firebase Authentication: 사용자 인증  
- Firestore (NoSQL):  
  - 사용자  
  - 반려동물  
  - 장소  
  - 리뷰 데이터 저장  
- Cloud Functions: 데이터 처리 및 API 연동  

### 3. AI 챗봇 (GPT API)
- OpenAI GPT API 기반  
- 동작 흐름:
  1. 사용자 질의 입력  
  2. 조건 추출  
  3. DB 필터링  
  4. 자연어 형태 응답 생성  

### 4. 데이터 처리
- 공공데이터 + 크롤링 데이터 수집  
- 데이터 정제 및 태그화  
- 규칙 기반 필터링 및 간단한 점수화 로직 적용

## Ⅴ. 기술 스택

- **Frontend**: Flutter, Dart  
- **Backend**: Firebase Auth, Firestore, Cloud Functions  
- **AI**: OpenAI GPT API  
- **Data Processing**: Python  
- **Infra**: Firebase 기반 서버리스 구조  
- **External API**: Kakao Map API, 공공데이터포털 API
